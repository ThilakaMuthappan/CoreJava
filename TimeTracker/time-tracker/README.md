# Time Tracker App

## Usage

### Start logging Time
`time-tracker <task-name> start`
`time-tracker <task-name> <category-name> start`

Note: If the task name is already in progress this will be a no-op

### Stop logging time

`time-tracker <task-name> stop`

### Show Time
`time-tracker task report`
Show the tasks and time spent of each task

`time-tracker category report`
Shows the categories and time spent on each category
